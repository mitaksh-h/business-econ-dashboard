# Business Economics & Insights Dashboard

A universal business profitability & insights dashboard with a recommendation co-pilot.  
It calculates unit economics, tracks customer metrics, and generates actionable recommendations.

## Tech Stack
- Python (pandas, numpy)
- Streamlit
- scikit-learn / statsmodels
- Docker

## Structure
- `data/` → synthetic dataset
- `scripts/` → data generator scripts
- `app/` → Streamlit dashbo
